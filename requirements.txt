# ComfyUI-SortList Requirements

# ============================================
# Python Packages: NONE REQUIRED
# ============================================
# This node uses only Python built-in modules:
# - os
# - subprocess  
# - math
# - concurrent.futures

# ============================================
# System Binaries: REQUIRED
# ============================================
# The following system binaries MUST be installed:
# - ffmpeg (for video processing)
# - ffprobe (for video metadata detection)

# NOTE: These CANNOT be installed via pip!
# They are system-level binaries that must be installed separately.

# ============================================
# Installation Instructions
# ============================================

# Google Colab (Usually Pre-installed):
# Run this to verify (should already exist):
#   !ffmpeg -version
#   !ffprobe -version
#
# If not installed, run:
#   !apt-get update -qq && apt-get install -y ffmpeg

# Local Ubuntu/Debian:
#   sudo apt-get update
#   sudo apt-get install ffmpeg

# Local macOS:
#   brew install ffmpeg

# Local Windows:
#   Download from: https://ffmpeg.org/download.html
#   Add to PATH after installation
